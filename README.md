## 🌟 Overview

The **Premium Paywall Calculator** is a state-of-the-art frontend application built with React and Vite. It demonstrates an elegant implementation of a "freemium" user model, smoothly intercepting user interactions with a beautiful paywall that offers different subscription plans and a seamless mock checkout experience.

Perfect for showcasing modern glassmorphism UI design, robust React state management, and standard component architectures.

## 🚀 Features

- **Gorgeous Glassmorphism UI:** Built completely with vanilla CSS, featuring deep dark mode aesthetics, dynamic radial gradients, translucent backgrounds, and stunning hover effects.
- **Dynamic Feature Lock:** Attempts to calculate mathematical equations accurately capture state, but beautifully blur out (`???`) the results in the interface until the user successfully upgrades.
- **Multi-tiered Subscription Plans:** 
  - `Basic Plan` (Unlock 1 answer)
  - `Premium Plan` (Unlimited math unlocks + A golden **PRO** Badge)
- **Secure Checkout Interface:**
  - **Card Payment Tab:** Fully interactive, editable input fields for your Card Number, Expiry, and dynamically masked CVV data.
  - **UPI Tab:** Complete UPI flow featuring a field for a UPI ID directly alongside a scannable QR Code element.

## 🛠️ Local Installation

If you would like to run and modify this application locally on your own machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mmaneesh007/calculator.git
   cd calculator
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Spin up the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open your browser and navigate to `http://localhost:5173`.
